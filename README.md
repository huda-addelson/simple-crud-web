## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/huda-addelson/simple-crud-web.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

- Run as development
  ```sh
  npm run dev
  ```
- Build the app
  ```sh
  npm run build
  ```
- Run as production
  ```sh
  npm run start
  ```
- Lint the code with ESLint
  ```sh
  npm run lint
  ```
